## How Credits Work

Each API call costs a fixed number of credits. **1 credit = $0.01 USD.** New accounts start with **500 free credits ($5)**.

## Costs by Endpoint

| Endpoint | Method | Credits | Cost |
|----------|--------|---------|------|
| `/api/v1/ask` | POST | 5 | $0.05 |
| `/api/v1/ask` (deep mode) | POST | 20 | $0.20 |
| `/api/v1/ask/stream` | POST | 5 / 20 | Same as sync |
| `/api/v1/research/search` (1-20 results) | POST | 1 | $0.01 |
| `/api/v1/research/search` (21-50 results) | POST | 2 | $0.02 |
| `/api/v1/research/search` (51-100 results) | POST | 3 | $0.03 |
| `/api/v1/research/quick` | POST | 1 | $0.01 |
| `/api/v1/citations/resolve` | GET | 1 | $0.01 |
| `/api/v1/citations/cases/search` | GET | 1 | $0.01 |
| `/api/v1/citations/cases/lookup` | GET | 1 | $0.01 |
| `/api/v1/citations/cases/network` | GET | 2 | $0.02 |

Credit cost is per call, regardless of result size.

## Checking Your Balance

View remaining credits in your [developer settings](https://app.vaquill.ai/developer?tab=credits). When credits reach zero, API calls return `403`.

## Notes
