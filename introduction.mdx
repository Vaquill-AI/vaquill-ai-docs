## Coverage

- **20M+ cases** across multiple courts and jurisdictions
- **Citation formats:** SCC, AIR, INSC, and more
- **50+ file formats** including PDF, DOCX, scanned documents (OCR)

## Getting Started

### Use the Platform

<Steps>
  <Step title="Create an account">
    Sign up at [app.vaquill.ai](https://app.vaquill.ai). Free plan includes 25 monthly messages and 100 pages.
  </Step>
  <Step title="Upload a document">
    Drag and drop a file into any conversation.
  </Step>
  <Step title="Ask a question">
    Type a question in plain language. Each answer includes source citations with page references.
  </Step>
</Steps>

### Use the API

<Steps>
  <Step title="Get an API key">
    Generate a key from your [dashboard settings](https://app.vaquill.ai/settings).
  </Step>
  <Step title="Make your first request">
    Search for a case:

    ```bash
    curl -X POST https://api.vaquill.ai/api/v1/research/search \
      -H "Authorization: Bearer vq_key_..." \
      -H "Content-Type: application/json" \
      -d '{"query": "right to privacy", "limit": 5}'
    ```
  </Step>
  <Step title="Explore the endpoints">
    Browse the full [API Reference](/api-reference) for case search, citation resolution, and citation network traversal.
  </Step>
</Steps>
