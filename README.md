# Vaquill AI Docs

Documentation and API reference for [Vaquill AI](https://vaquill.ai) — legal research platform and case law API.

**Docs:** [vaquill.ai/docs](https://vaquill.ai/docs)

## Local Development

```bash
npx mintlify dev
```

Serves at `http://localhost:3000` with hot reload.
